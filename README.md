# Математический анализ — семинары

Конспекты семинаров 4-го семестра.

## Структура

- `Seminars/` — markdown-файлы семинаров (Sem1.md, Sem2.md, …)
- `build-pdf.ps1` — сборка PDF

## Сборка PDF

```powershell
.\build-pdf.ps1
```

Требуется: **Pandoc**, **XeLaTeX**, шрифт Cambria. Результат: `Seminars.pdf`.
