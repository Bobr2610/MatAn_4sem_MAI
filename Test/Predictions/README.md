# Обзор вариантов контрольной работы

Создано **20 вариантов** контрольной работы по математическому анализу на основе ваших записей семинаров и демо-версии.

## Структура каждого варианта

Все варианты следуют единой структуре из 6 заданий:

### Задание 1 — Знакоположительные ряды
Исследование на сходимость с использованием различных признаков:
- Признак Даламбера
- Радикальный признак Коши
- Интегральный признак Коши
- Признак Раабе-Дюамеля
- Признак Гаусса
- Предельный признак сравнения
- Признак сравнения

### Задание 2 — Знакопеременные ряды
Определение типа сходимости (абсолютная/условная/расходимость):
- Знакочередующиеся ряды (признак Лейбница)
- Ряды с признаком Дирихле
- Ряды с признаком Абеля

### Задание 3 — Функциональные/степенные ряды
Два типа заданий:
- **Тип А**: Исследование функционального ряда на равномерную сходимость (признаки Вейерштрасса, Дирихле, Абеля)
- **Тип Б**: Исследование степенного ряда (центр, радиус, области сходимости)

### Задание 4 — Разложения Тейлора/Маклорена
Запись разложения с остаточным членом:
- В форме Лагранжа
- В форме Пеано

Функции: $e^x$, $\sin x$, $\cos x$, $\ln x$, $\sqrt{x}$, $\sqrt[3]{x}$, $(1+x)^p$, $\arctan x$, и др.

### Задание 5 — Ряды Фурье
Три типа заданий:
- Полный ряд Фурье на $\langle -L, L \rangle$
- Ряд "по косинусам" на $\langle 0, L \rangle$
- Ряд "по синусам" на $\langle 0, L \rangle$

### Задание 6 — Теоретический вопрос
Темы из списка `Themes.md`:
- Определения (сходимость, равномерная сходимость, степенные ряды)
- Признаки сходимости
- Примеры рядов с различными свойствами
- Свойства степенных рядов

## Особенности вариантов

- **Вариант 20** максимально близок к демо-версии (задачи 1, 2, 4, 5, 6 совпадают)
- Варианты используют примеры и методы из семинаров 1-9
- Задачи подобраны с учетом разнообразия методов решения
- Теоретические вопросы охватывают весь список из `Themes.md`

## Рекомендации по использованию

1. Варианты можно использовать для тренировки перед контрольной
2. Каждый вариант содержит задачи разного уровня сложности
3. Для полной подготовки рекомендуется решить 3-5 вариантов
4. Обратите внимание на задачи с указанием конкретного признака (как в демо-версии)
