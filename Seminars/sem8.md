## Признаки равномерной сходимости функциональных рядов

*   **Теорема (Вейерштрасса):**
    Пусть:
    1) Дан числовой ряд $\sum_{n=1}^{\infty} u_n(x)$, определенный как минимум на множестве $D$.
    2) Дан сходящийся числовой ряд $\sum_{n=1}^{\infty} c_n$, причем все $c_n \ge 0$.
    3) $\forall n \; \forall x_0 \in D \quad (|u_n(x_0)| \le c_n)$
       *(в этом случае ряд $\sum_{n=1}^{\infty} c_n$ называется мажорирующим для $\sum_{n=1}^{\infty} u_n(x)$)*

    Тогда ряд $\sum_{n=1}^{\infty} u_n(x)$ равномерно сходится на множестве $D$.

*   **Пример 1.** $\sum_{n=1}^{\infty} \frac{\sin(nx)}{n^2}$
    Исследовать на равномерную сходимость на множестве $D = [-\pi; \pi]$
    *Решение:*
    $|u_n(x)| = \left| \frac{\sin(nx)}{n^2} \right| \le \frac{1}{n^2} = c_n$  *(указано, что $|\sin(nx)| \le 1$ для $x \in [-\pi; \pi]$)*
    $\forall n \; \forall x_0 \in D \quad (|u_n(x_0)| \le c_n)$
    $\Rightarrow \sum_{n=1}^{\infty} \frac{\sin(nx)}{n^2}$ равномерно сходится на множестве $D$.

*   **Пример 2.** $\sum_{n=1}^{\infty} x^n e^{-\frac{nx^2}{2}} \quad D = (0; +\infty)$
    $u_n' = \left( x^n e^{-\frac{nx^2}{2}} \right)' = n x^{n-1} \cdot e^{-\frac{nx^2}{2}} + x^n \cdot e^{-\frac{nx^2}{2}} \cdot \left( -\frac{n \cdot 2x}{2} \right) =$
    $= x^{n-1} e^{-\frac{nx^2}{2}} (n - nx^2) = n x^{n-1} e^{-\frac{nx^2}{2}} (1 - x^2) = 0 \Rightarrow x^2 = 1 \Rightarrow x = \pm 1$

---

*(продолжение Примера 2)*
$u_n' = \left( x^n e^{-\frac{nx^2}{2}} \right)' = n x^{n-1} e^{-\frac{nx^2}{2}} + x^n e^{-\frac{nx^2}{2}} \cdot (-nx) = n x^{n-1} e^{-\frac{nx^2}{2}} (1 - x^2)$
*(Нарисована числовая прямая для производной, точка 1, знаки '+' слева, '-' справа)*
$\Rightarrow c_n = u_n(1) = e^{-\frac{n}{2}}$

Итак $\forall x \in D = (0; +\infty) \quad |u_n(x)| \le u_n(1) = e^{-\frac{n}{2}} = c_n$
Заметим, что $\sum_{n=1}^{\infty} e^{-\frac{n}{2}}$ сходится по радикальному признаку Коши.
По признаку Вейерштрасса исходный ряд равномерно сходится на $D$.

*   **Теорема (признак Дирихле равномерной сходимости функционального ряда)**
    Пусть
    1) дан ряд $\sum_{n=1}^{\infty} a_n(x)b_n(x)$, который определен как минимум на множестве $D$
    2) Для каждого фиксированного $x_0 \in D$
       либо $a_1(x_0) \le a_2(x_0) \le a_3(x_0) \le ...,$
       либо $a_1(x_0) \ge ...$
    3) $a_n \overset{D}{\rightrightarrows} 0$, т.е. $\forall \varepsilon > 0 \; \exists N \; \forall n \ge N \; \forall x \in D \; (|a_n(x) - 0| < \varepsilon)$
    4) Все суммы $(b_1(x) + ... + b_n(x))$ равномерно ограничены на $D$,
       т.е. $\exists C > 0 \; \forall N \; \forall x_0 \in D \; (|b_1(x_0) + ... + b_N(x_0)| \le C)$

    Тогда $\sum_{n=1}^{\infty} a_n(x)b_n(x)$ сходится равномерно на $D$.
